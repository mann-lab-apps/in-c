# 공연 홍보 반복 작업 시간 절감 가설 검증 계획

작성일: 2026-07-16

## 가설

소규모 클래식 공연 주최자는 공연명, 작품 설명, 출연자 소개, 예매 링크, 포스터 문구를
예매처, SNS, 공연장 홈페이지, 개인 메시지에 반복해서 다시 쓰며 시간을 쓴다.
in C Concert Preview가 작품 맥락과 듣기 질문을 한 번 정리해주면 이 반복 시간을
줄일 수 있다.

## 홍보 단계별 시간 가설

| 단계 | 반복 작업 | 측정 질문 |
| --- | --- | --- |
| 공연 정보 정리 | 제목, 일시, 장소, 예매 링크 확인 | 한 공연 정보를 처음 정리하는 데 몇 분이 드나요 |
| 작품 설명 작성 | 작품 소개와 연주 포인트 작성 | 같은 작품 설명을 몇 번 다시 쓰나요 |
| 채널별 게시 | SNS, 블로그, 예매처, 단체 메시지 | 채널별로 문구를 바꾸는 데 얼마나 걸리나요 |
| 문의 대응 | 위치, 시간, 프로그램, 예매 관련 답변 | 반복해서 받는 질문은 무엇인가요 |
| 공연 후 정리 | 후기, 사진, 다음 공연 연결 | 공연 후 관객 반응을 어디에 남기나요 |

## 인터뷰 질문

- 최근 공연 하나를 떠올렸을 때 홍보 문구를 몇 곳에 올렸나요?
- 한 곳에 올린 내용을 다른 채널에 맞춰 다시 쓰는 데 얼마나 걸렸나요?
- 작품 설명, 출연자 소개, 예매 안내 중 가장 반복되는 항목은 무엇인가요?
- 관객이 공연 전에 가장 자주 묻는 질문은 무엇인가요?
- in C가 작품별 듣기 질문과 짧은 프리뷰를 제공하면 어느 채널에서 재사용할 수 있나요?
- 절감 시간을 돈으로 환산하기보다, 덜 부담스러운 작업으로 느껴질 조건은 무엇인가요?

## 판정 기준

- 3명 이상이 같은 반복 작업을 언급하면 Concert Preview 파일럿 입력 필드에 반영한다.
- 절감 시간이 10분 미만이어도 주최자가 "다시 쓰기 부담"을 강하게 말하면 정성 근거로
  남긴다.
- 티켓 판매 증가나 광고 효율 개선은 이 가설의 직접 완료 기준으로 쓰지 않는다.
