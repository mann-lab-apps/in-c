## 목적

제한사항을 카탈로그로 관리한다.

아래 항목은 후보이며, 실제 수치 인용 전에는 URL, 기준연도, 조회일, 이용 조건을 확인해야
한다.

## 출처 등록 체크리스트

- [ ] 공식 출처와 표본 출처를 구분했다.

- [ ] 사업계획서에 직접 쓸 수 있는지와 제품 운영 참고용인지 구분했다.

- [ ] SNS, 커뮤니티, 인터뷰 자료라면 개인정보와 원문 복제를 제거했다.

## 자동화 전 확인할 것

- 공개 API 또는 공식 다운로드가 있는가?
- robots와 이용약관이 자동 수집을 허용하는가?
- 로그인이나 우회 접근이 필요한가?
- 원문 복제가 아니라 링크/메타데이터 기록으로 충분한가?
- 수집 빈도를 낮춰도 운영 목적을 달성할 수 있는가?
