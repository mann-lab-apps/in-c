# 점음표와 점쉼표

## 모델

`Duration.dots`는 boolean이 아니라 0부터 3까지의 정수다. 홑점, 겹점,
셋점은 기본 음가에 각각 다음 비율을 적용한다.

- 홑점: `3/2`
- 겹점: `7/4`
- 셋점: `15/8`

음표와 쉼표는 같은 duration 계산을 사용한다. 지원 범위를 벗어난 점
개수는 score-core 경계에서 거부한다.

## 편집

점 추가와 제거는 기존 `buildRhythmEditCommand()`를 사용한다.

- 점 추가로 길이가 늘면 바로 뒤의 연속된 쉼표를 소비한다.
- 마지막 쉼표를 일부만 소비하면 남은 구간을 표기 가능한 쉼표로
  다시 분해한다.
- 점 제거로 길이가 줄면 해제된 구간을 쉼표로 채운다.
- 다른 음표를 침범하거나 마디를 넘는 증가는 거부한다.
- 타이 시작 음의 점 제거는 타이 사이에 쉼표를 만들기 때문에 거부한다.

입력 상태에서는 점 개수를 먼저 선택한 뒤 음표나 쉼표를 입력한다.
`.`은 점 추가, `,`는 점 제거 단축키다.

## 출력

VexFlow은 `Duration.dots` 개수만큼 `Dot` modifier를 붙인다. 재생은 같은
duration tick 계산을 사용하며, MusicXML은 점마다 별도의 `<dot/>`을
내보내고 가져올 때 개수를 복원한다.
