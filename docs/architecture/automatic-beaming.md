# 박자 기반 자동 빔

## 기본 규칙

빔 그룹은 VexFlow 객체를 만들기 전에 score-core의 이벤트 tick과 박자표만으로
계산한다.

- 4/4와 3/4는 4분음표 박 단위로 그룹을 나눈다.
- 6/8은 8분음표 세 개, 즉 점4분음표 박 단위로 그룹을 나눈다.
- 8분음표 이하의 음표만 자동 빔 대상이다.
- 쉼표, 4분음표 이상의 긴 음가, 시간 gap과 박 경계에서 그룹을 끊는다.
- 그룹에는 최소 두 음표가 있어야 한다.
- 빔은 마디 경계를 넘지 않는다.

`createBeamGroups()`는 event ID 그룹만 반환하며 VexFlow에 의존하지 않는다.
렌더러는 이 결과를 해당 `StaveNote`에 매핑하고 `Beam`을 생성한다. duration이나
position이 바뀌면 score 재렌더링 과정에서 그룹을 다시 계산한다.

## 확장 경계

현재 규칙은 자동 빔만 다룬다. 사용자가 직접 지정한 빔 시작·종료와 보조 빔
분리는 향후 별도 relation 또는 engraving override 모델로 추가하며, 자동 계산
결과를 score-core에 저장하지 않는다.
