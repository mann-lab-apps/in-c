# 음높이와 임시표 문맥

## 모델 원칙

`Pitch.alter`는 조표와 무관한 실제 반음 변경값이다. 렌더링, 재생,
MusicXML 내보내기는 모두 같은 유효 음높이 계산을 사용한다.

- `step`: 기보된 음이름
- `octave`: 기보된 옥타브
- `alter`: `-2`부터 `2`까지의 실제 반음 변경값
- `undefined`: 현재 마디의 임시표 문맥 또는 조표를 상속

이 구조는 음이름 표기를 보존하면서도 재생과 내보내기의 실제 음높이를
일치시킨다.

## 유효 임시표 계산

한 음의 유효 `alter`는 다음 순서로 결정한다.

1. 음에 명시된 `alter`
2. 같은 마디에서 앞서 나온 같은 `step`과 `octave`의 임시표
3. 마디의 조표

임시표 문맥은 마디 경계에서 초기화되며 옥타브별로 독립적이다. 화면과
MusicXML의 임시표 기호는 유효 `alter`가 직전 문맥을 바꿀 때만 표시한다.

## 입력과 편집

문자 음이름 입력은 직전 음과 가장 가까운 옥타브를 선택한다. 입력 전
지정한 플랫, 제자리표, 샤프는 다음 음 하나에 적용한 뒤 해제된다.

선택 음표에는 다음 이동을 지원한다.

- 온음계 이동: `ArrowUp`, `ArrowDown`
- 반음계 이동: `Alt+ArrowUp`, `Alt+ArrowDown`
- 옥타브 이동: `Shift+ArrowUp`, `Shift+ArrowDown`

모든 음높이 변경은 score command로 실행하므로 일반 편집과 동일하게
undo/redo할 수 있다.

## 확장 경계

현재 범위는 단성부와 기본 조표의 임시표 문맥이다. 이명동음 자동 표기,
조표 변경 지점, courtesy accidental, 미분음 accidental은 별도 정책으로
확장한다.
