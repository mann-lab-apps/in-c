# Community 학습 후보 파트너 파일럿 제안 기준

작성일: 2026-07-16

## 목적

Community 안의 학습 후보 파일럿은 수업 신청이나 결제를 처리하는 기능이 아니라,
교육자와 입문자가 작품, Columns, 악보, 관련 인물 맥락을 통해 수업 전후 질문을
더 쉽게 나눌 수 있는지 확인하는 정보형 파일럿이다.

## 파일럿 대상

| 대상 | 적합한 이유 | 우선순위 |
| --- | --- | --- |
| 클래식 감상 입문 수업 | Columns 질문과 바로 연결된다 | P1 |
| 악보 읽기/기초 이론 수업 | Compositions와 Chromatics 예제가 연결된다 | P1 |
| 공연 전 해설 수업 | Concert Preview와 연결된다 | P2 |
| 개인 레슨 홍보 | 결제/신청 경계가 필요하다 | P2 |

## 포함 범위

- 클래스 제목, 대상, 난이도, 공개 소개
- 관련 작품, Columns, Compositions, 관련 인물 연결
- 수업에서 쓸 수 있는 듣기 질문 또는 악보 예제
- 신청/결제 기능이 없다는 상태 문구

## 제외 범위

- 결제, 예약, 출석 관리, 환불
- 학생 개인정보와 연락처 저장
- 강의 원문, 유료 교재, 저작권 불명확 자료 업로드
- 수업 품질 보증이나 강사 자격 심사

## 성공 기준

| 질문 | 확인 방법 |
| --- | --- |
| 교육자가 수업 설명을 더 쉽게 재사용하는가 | intake 후 1회 회고 |
| 입문자가 수업 전에 들어볼 질문을 이해하는가 | Community 학습 후보 상세 피드백 |
| 작품/악보/Columns 연결이 실제 수업 준비에 도움이 되는가 | 교육자 인터뷰 |

## 사이트 문구 기준

- "신청 가능", "예약", "결제"처럼 실제 기능이 있는 것으로 보이는 표현은 쓰지 않는다.
- 후보 데이터는 "파일럿 검토", "정보형 프리뷰"로 표시한다.
- 공개 링크가 없으면 문의를 유도하지 않고 현재 준비 중인 정보로 둔다.
