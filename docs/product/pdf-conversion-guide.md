# PDF 변환 안내

Chromatics의 `PDF 변환`은 현재 화면의 악보를 읽기와 공유에 알맞은 PDF 파일로
저장합니다. PDF는 편집 원본이 아닙니다. 나중에 다시 편집하거나 다른 사보 도구와
주고받으려면 먼저 `MusicXML로 저장`으로 `.musicxml` 파일을 함께 보관하세요.

## PDF를 만드는 순서

1. Chromatics 데스크탑 앱에서 새 악보를 만들거나 MusicXML 파일을 엽니다.
2. 제목과 악보 내용을 확인합니다. 제목은 제안되는 PDF 파일명에 사용됩니다.
3. 파일 도구에서 `PDF 변환`을 누릅니다.
4. 저장할 폴더와 파일명을 고른 뒤 저장합니다.
5. 완료 안내에 나온 PDF 파일을 열어 모든 마디와 기호가 보이는지 확인합니다.

저장 창에서 취소하면 파일을 만들지 않으며 성공 안내도 표시하지 않습니다.

## MusicXML과 PDF의 차이

| 목적 | 사용할 파일 | 현재 역할 |
| --- | --- | --- |
| 악보를 다시 편집하거나 다른 도구와 주고받기 | MusicXML | 편집 가능한 교환용 원본 |
| 완성된 악보를 읽거나 인쇄·공유하기 | PDF | 현재 화면을 바탕으로 만든 결과물 |

Chromatics는 PDF 파일 가져오기나 PDF 내용을 다시 편집하는 기능을 제공하지
않습니다. PDF만 남기지 말고 MusicXML 파일도 보관하세요. 현재 안정 지원 범위는
단일 파트·단성부 중심이며, 자세한 제한은
[MusicXML MVP 지원 범위](../musicxml-mvp.md)에서 확인할 수 있습니다.

## 실패했을 때 확인할 순서

1. 저장할 폴더에 파일을 만들 권한과 남은 저장 공간이 있는지 확인합니다.
2. 다른 폴더와 짧은 파일명으로 다시 시도합니다.
3. `MusicXML로 저장`으로 편집 원본을 먼저 보관합니다.
4. 앱을 다시 열고 같은 MusicXML 파일로 PDF 변환을 다시 시도합니다.

계속 실패하면 GitHub 이슈에 운영체제, 앱 버전, 오류 문구, 재현 순서를 남깁니다.
악보에 개인정보나 공개하면 안 되는 내용이 있으면 파일 자체는 공개 이슈에
첨부하지 않습니다. 릴리즈 후보를 점검하는 담당자는
[작은 악보 완성 수동 QA](../releases/manual-score-completion-qa.md)의 9단계를 함께
확인합니다.
