# Creator 프로필 intake 질문지

작성일: 2026-07-16

## 목적

Creator 프로필은 작곡가, 연주자, 지휘자, 기획자, 교육자를 작품, 공연, Columns,
Classes와 연결하기 위한 공개 정보 표면이다. 이 질문지는 공개 가능한 최소 정보를
받되, 개인정보와 비공개 협력 정보를 레포에 저장하지 않기 위한 기준이다.

## 필수 질문

| 질문 | 용도 | 공개 여부 |
| --- | --- | --- |
| 공개 이름 또는 활동명은 무엇인가요? | 프로필 제목 | 공개 |
| 주 역할은 무엇인가요? | 역할 taxonomy | 공개 |
| 연결할 작품, 공연, Columns, Classes가 있나요? | 관계 데이터 | 공개 가능한 항목만 |
| 한 문장 소개를 어떻게 쓰면 좋을까요? | 목록 카드 | 공개 |
| 현재 공개 가능한 공식 링크가 있나요? | 외부 링크 | 공개 |

## 선택 질문

| 질문 | 용도 |
| --- | --- |
| 입문자가 먼저 들어볼 작품이나 질문이 있나요? | Columns/Works 연결 |
| 예정 공연이나 수업과 연결하고 싶은 내용이 있나요? | Concerts/Classes 연결 |
| 공개 프로필에서 제외해야 할 활동이 있나요? | 노출 경계 |
| in C에서 다루면 좋은 인터뷰 주제가 있나요? | 콘텐츠 backlog |

## 받지 않는 정보

- 개인 전화번호, 비공개 이메일, 주민등록번호, 계좌, 주소
- 단체 채팅방 내용, 내부 출연료, 계약 조건
- 권리 상태가 불명확한 사진, 악보, 음원, 포스터 원본
- 동의받지 않은 제3자 실명과 연락처

## 최소 필드 기준

| 필드 | 값 예시 | 비고 |
| --- | --- | --- |
| `name` | 공개 활동명 | 필수 |
| `roles` | composer, performer, educator, organizer | 복수 가능 |
| `summary` | 한 문장 소개 | 필수 |
| `relatedWorks` | work slug | 공개 데이터만 |
| `relatedConcerts` | concert slug | 후보/프리뷰 상태 구분 |
| `relatedClasses` | class slug | 신청/결제 없음 상태 구분 |
| `externalUrl` | 공식 사이트 또는 공개 프로필 | 선택 |
