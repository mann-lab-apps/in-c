Notice

개인정보·문의·운영 고지

이 문서는 현재 공개된 in C 정적 사이트와 Chromatics 다운로드 안내에 적용되는 최소 고지입니다. 계정, 결제, 커뮤니티 기능을 제공하기 전에는 별도의 약관과 개인정보 처리방침을 다시 검토합니다.

Analytics

Google Analytics 사용

in C는 공개 사이트의 페이지 조회, 다운로드 클릭, Columns 읽기 완료, Compositions 열기와 같은 제품 개선용 이벤트를 Google Analytics 4로 확인합니다. 이벤트 payload에는 공개 콘텐츠 제목, slug, 난이도, 파일명, 클릭 위치처럼 사이트 화면에 이미 드러나는 정보만 보냅니다.

이메일, 이름, 전화번호, 주소, 계정 ID, 앱 내부 악보 편집 내용은 사이트 analytics 이벤트로 보내지 않습니다. analytics가 비활성화되거나 로드에 실패해도 페이지 탐색과 다운로드 링크는 그대로 동작해야 합니다.

Feedback

피드백과 문의

Columns 피드백 UI는 자유 텍스트가 아니라 미리 정한 선택지 값만 전송합니다. 공개 문의와 버그 제보 채널은 계정·결제·커뮤니티 기능을 열기 전에 별도로 고지합니다.

제보에는 재현 순서, 사용한 운영체제, 앱 버전, 화면 캡처처럼 문제를 확인하는 데 필요한 정보만 남겨 주세요. 개인 연락처, 민감한 악보, 비공개 파일은 공개 채널에 올리지 않는 것을 권장합니다.

Contact

문의 경로

현재 공개 사이트는 계정, 결제, 커뮤니티 기능을 제공하지 않습니다. 운영 문의와 버그 제보를 받을 상시 채널은 공개 기능을 확장하기 전에 별도로 고지합니다.
