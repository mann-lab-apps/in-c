Chromatics 앱

in C의 무료 악보 편집 도구입니다. 제품군 전체가 아니라 창작 표면입니다.

Chromatics 앱

Chromatics는 단성부 악보 작성에 집중합니다

음표·쉼표 입력

악보를 처음 다루는 사람도 시작하기 쉽게 입력 흐름을 다듬습니다.

기본 리듬 표기

점음표, 타이, 셋잇단음표와 자동 빔을 단성부 범위에서 다룹니다.

MusicXML 흐름

간단한 악보를 가져오고 내보내며 다른 사보 도구와 연결할 수 있습니다.

현재 제한 사항

  • 화음, 코드 심벌, 가사 입력은 MVP 이후 기능입니다.
  • 일부 빔 기울기와 높은/낮은 음역 렌더링은 계속 안정화 중입니다.

  • 문제가 보이면 GitHub Issues에 재현 순서와 화면 캡처를 남겨 주세요.
